## Getting Started

### Prerequisites

- Quarto installed ([quarto.org](https://quarto.org))
- Git installed
- GitHub account

### Installation

1. Clone this repository:
```bash
git clone https://github.com/parmindersingh253514/qmd-eportfolio.git
cd qmd-eportfolio
```

2. Preview the site locally:
```bash
quarto preview
```

3. Render the site:
```bash
quarto render
```
